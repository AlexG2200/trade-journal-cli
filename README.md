# Trade Idea & Stock Journal (V1)

This project is a **Python command-line (CLI) journal** designed for traders to log:
- **Weekly trade ideas** (ex: Sunday scans)
- **Aftermarket scans** (plans + key levels for the next trading day)

---

## Why I Built This
As a trader, I wanted a simple and fast way to journal my trade ideas and next-day plans without needing a full app or complicated setup.  
This tool lets me build consistency, track my thinking, and review my ideas later.

---

## Features (V1)
- Menu-based CLI program (easy to use)
- Auto date + time stamps for every entry
- Two journal modes:
  1. Weekly trade idea journal
  2. Aftermarket scan journal
- Saves entries into a local file: `journal.txt`

---

## Requirements
- Python 3.8+ recommended
- No external libraries required
